Love2D Template
===
This is a template that is meant to be appropriate for any Love2D game. It includes some structure, helpers and libraries.


## Classes
* Map
* MapTile
* MapEntity
* Crew
* Survivor (potentially just a state of crew?)
* Enemy
* Tower
* Gun
* Wall
* Door/Gate?
* Salvage

## Mixins
* Repairable
* Movable
